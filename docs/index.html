Core Features

专业级功能，完全免费

🖥

多模式屏幕录制

支持整个屏幕、应用窗口或单个浏览器标签页三种模式，灵活适配教程录制、Bug 复现、产品演示。

区域录制

开始前拖拽绘制矩形选区，只录制指定区域。Retina 屏自动处理 DPR 坐标换算，像素级精准。

devicePixelRatio
📷

摄像头画中画

圆形摄像头泡泡叠加于屏幕角落，支持自由拖动调整位置，直接合入录制画面，无需后期叠加。

Canvas PiP
🔊

多路音频混音

AudioContext 同时捕获系统声音和麦克风，两路实时混合输出，无需后期合并，一次录好。

AudioContext API
✏️

实时屏幕标注

录制中实时绘制画笔、荧光笔、箭头、圆形高亮，标注直接合入画面，支持 Ctrl+Z 撤销。

暂停 / 继续

录制中随时暂停恢复，配合全局快捷键无需切窗口，Service Worker 状态持久化保证稳定。

3 秒倒计时蒙层

选择屏幕后自动显示全屏 3 秒倒计时，确保录制从第一帧开始内容完整，告别手忙脚乱。

Overlay Injection
🎯

悬浮工具栏

Under The Hood

纯原生 Web API，
零第三方依赖

SnapCast 基于 Chrome Manifest V3，全部由原生 JavaScript (ES6+) 驱动，不引入任何 npm 包和运行时库。你看到的每个功能，都是浏览器 API 的直接映射。

getDisplayMedia()

屏幕 / 窗口 / 标签页捕获

getUserMedia()

摄像头 & 麦克风采集

canvas.captureStream()

Canvas → 视频流转换

MediaRecorder

WebM 格式视频编码录制

AudioContext

多路音频节点实时混音

PointerEvent

标注工具触控 / 鼠标事件

chrome.downloads

本地文件写入下载

chrome.commands

全局快捷键注册与监听

Quick Start

3 步开始录制

1

下载扩展源码

从 GitHub 克隆仓库或下载 ZIP 解压。无需 npm install，无任何构建步骤，开箱即用。

2

加载到 Chrome

访问 chrome://extensions/，开启「开发者模式」，点击「加载已解压的扩展程序」并选择目录。

3

点击图标，开始录制

工具栏点击 SnapCast 图标，配置麦克风 / 摄像头，点击「开始录制」并选择屏幕范围即可。
